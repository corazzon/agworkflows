// turbo-all

1. **[환경]** `streamlit`, `plotly`, `pandas`, `scipy`, `koreanize_matplotlib` 등 필수 라이브러리 설치를 확인합니다.
2. **[대시보드 구축]** 
    - `src/dashboard.py` 파일을 생성합니다.
    - **데이터 로드**:
        - 사이드바에서 데이터 파일 업로드 (`st.file_uploader`) 및 데이터 미리보기(head) 기능 구현.
    - **[핵심] 5가지 이상 분석 메뉴(Tabs) 구현 요건**:
        1. **데이터 개요 및 품질 (Overview)**: 기초 통계량, 결측치 분포, 이상치 탐지 등 (그래프 5개 이상).
        2. **단변량 분석 (Univariate)**: 컬럼별 히스토그램, 박스플롯, 커널 밀도 추정(KDE) 등 (그래프 5개 이상).
        3. **이변량/상관 분석 (Bivariate)**: 산점도, 상관계수 히트맵, 범주별 수치 비교 박스플롯 등 (그래프 5개 이상).
        4. **시계열/트렌드 (Timeseries)**: (날짜 데이터 존재 시) 추세선, 이동평균, 월별/요일별 패턴 등 (그래프 5개 이상).
        5. **고급 교차 분석 (Pivot & Crosstab)**: 사용자가 선택한 컬럼으로 피봇 테이블과 교차표를 생성하고 이를 히트맵이나 누적 막대로 시각화 (그래프 5개 이상).
    - **각 섹션 공통 필수 요소**:
        - **Graph**: Plotly를 이용한 인터랙티브 차트.
        - **Table**: 해당 그래프의 근거가 되는 **교차표(Crosstab)** 또는 **피봇 테이블(Pivot Table)** 데이터 표시.
        - **Interpretation**: "이 그래프는 ~를 보여주며, ~한 인사이트를 얻을 수 있습니다"와 같은 **자료 해석 방법/가이드** 텍스트 포함.
3. **[실행]** `streamlit run src/dashboard.py` 명령어를 실행합니다.
4. **[안내]** "브라우저에서 5가지 심층 분석 주제가 포함된 대시보드를 확인하세요."라고 안내합니다.
